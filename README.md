## About Narration

Narration was created by, and is maintained by [Nuno Maduro](https://github.com/nunomaduro), and is lightweight and modular micro-framework for developing fast and powerful web applications.

> **Note:** This repository contains the http component code of the Narration framework. If you want to build an application using Narration, visit the main [Narration repository](https://github.com/narration/narration).
